# Changelog

Все значимые изменения в проекте ST Village будут документированы в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
и этот проект придерживается [Semantic Versioning](https://semver.org/lang/ru/).

## [17.0] - 2026-04-05

### Добавлено
- `.gitignore` для исключения временных и системных файлов
- `CHANGELOG.md` для отслеживания версий и изменений
- `LICENSE` файл (MIT License)
- Улучшенная структура проекта с директориями `docs/` и `examples/`
- Проверка занятости портов 80/443 перед установкой
- Конфигурируемые пути установки (не только `/root`)
- Улучшенная обработка ошибок при занятых портах

### Изменено
- Рефакторинг конфигурационного блока для лучшей читаемости
- Оптимизация функций проверки портов
- Улучшена документация в README.md

### Исправлено
- Проблема с установкой при занятых портах другими сервисами

---

## [16.0] - 2026-04-04

### Добавлено
- Интерактивные мастера настройки `.env` с валидацией
- Диагностика сервисов: health check, DNS, порты, SSL
- Безопасность: UFW файрвол, Fail2ban
- Telegram-уведомления о событиях
- Восстановление из бэкапов
- Миграция между серверами
- Управление Swap
- Полное удаление проекта
- Автоматическая генерация JWT-секретов при установке

### Изменено
- Улучшена панель управления с цветным выводом
- Расширена система бэкапов с ротацией

---
