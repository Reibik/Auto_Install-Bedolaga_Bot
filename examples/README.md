# Примеры конфигурации

Эта директория содержит примеры конфигурационных файлов для ST Village.

## Файлы

- `bot.env.example` — пример конфигурации бота
- `cabinet.env.example` — пример конфигурации кабинета
- `Caddyfile.example` — пример конфигурации Caddy

## Использование

1. Скопируйте нужный пример
2. Замените значения на свои
3. Поместите файл в соответствующую директорию на сервере

## Примечания

- Никогда не коммитьте реальные токены и секреты в Git
- Используйте сильные пароли и секреты
- Регулярно обновляйте конфигурацию
