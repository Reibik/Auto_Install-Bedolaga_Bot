# 🎉 ST Village v17.0 - Release Notes

## 📦 Что нового

### ✨ Новые возможности

1. **Проверка портов перед установкой**
   - Автоматическая проверка занятости портов 80 и 443
   - Детальная информация о конфликтующих процессах
   - Возможность продолжить установку или отменить

2. **Конфигурируемые пути установки**
   - Переменная `ST_VILLAGE_BASE_DIR` для изменения базовой директории
   - Переменные `ST_VILLAGE_HTTP_PORT` и `ST_VILLAGE_HTTPS_PORT` для портов
   - Больше гибкости при развертывании

3. **Улучшенная структура проекта**
   - Директория `docs/` с документацией
   - Директория `examples/` с примерами конфигураций
   - Примеры для `.env` файлов и Caddyfile

4. **GitHub Actions**
   - Автоматическая проверка синтаксиса скрипта
   - ShellCheck для выявления потенциальных проблем
   - CI/CD для повышения качества кода

### 📝 Файлы проекта

- `.gitignore` - исключение временных файлов
- `CHANGELOG.md` - история изменений
- `LICENSE` - MIT лицензия

- `examples/` - примеры конфигураций

### 🔧 Улучшения

- Рефакторинг конфигурационного блока
- Улучшенная обработка ошибок
- Более информативные сообщения об ошибках
- Лучшая организация кода

### 🐛 Исправления

- Проблема с установкой при занятых портах
- Улучшенная валидация конфигурации

## 📥 Установка

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/Reibik/Auto_Install-Bedolaga_Bot/main/st_village.sh)
```

## 🔄 Обновление с v16.0

Если у вас уже установлена версия 16.0, просто запустите самообновление через меню панели:

```bash
./st_village.sh
# Выберите: 9 → 11 (Самообновление скрипта)
```
